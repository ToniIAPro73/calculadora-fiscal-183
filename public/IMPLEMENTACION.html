📦 Archivos Generados

  • favicon.ico - Formato ICO con múltiples tamaños (16x16, 32x32, 48x48)
  • favicon-16x16.png - Ícono pequeño para pestaña del navegador
  • favicon-32x32.png - Ícono estándar de pestaña
  • apple-touch-icon.png - Ícono para dispositivos iOS (180x180)
  • android-chrome-192x192.png - Ícono para Android (192x192)
  • android-chrome-512x512.png - Ícono de alta resolución para Android (512x512)
  • mstile-150x150.png - Ícono para Windows tiles (150x150)
  • site.webmanifest - Manifiesto de aplicación web (Android)
  • browserconfig.xml - Configuración para Windows

🚀 Instrucciones de Implementación

1. Estructura de Carpetas

Coloca todos los archivos de favicon en la raíz del proyecto:

project/ ├── public/ │ ├── favicon.ico │ ├── favicon-16x16.png │ ├── favicon-32x32.png │ ├── apple-touch-icon.png │ ├── android-chrome-192x192.png │ ├── android-chrome-512x512.png │ ├── mstile-150x150.png │ ├── site.webmanifest │ └── browserconfig.xml

2. Etiquetas HTML en <head>

Agrega las siguientes líneas al apartado <head> de tu documento HTML:

<!-- Favicons --> <link rel="icon" type="image/x-icon" href="/favicon.ico"> <link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png"> <link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png"> <link rel="apple-touch-icon" href="/apple-touch-icon.png"> <link rel="manifest" href="/site.webmanifest"> <meta name="msapplication-config" content="/browserconfig.xml"> <meta name="theme-color" content="#001a33">

3. Configuración para Next.js 15

Si utilizas Next.js, coloca los archivos en public/ y agrega en app/layout.tsx:

<Head> <link rel="icon" href="/favicon.ico" sizes="any"> <link rel="apple-touch-icon" href="/apple-touch-icon.png"> <link rel="manifest" href="/site.webmanifest"> </Head>

🎨 Especificaciones Técnicas

  • Formato primario: PNG con transparencia RGBA
  • Formato alternativo: ICO (múltiples tamaños)
  • Método de redimensionamiento: LANCZOS (máxima calidad)
  • Calidad de compresión: 95% (balance entre calidad y tamaño)
  • Color de tema: #001a33 (azul oscuro profundo)
  • Soporte de imagen maskable: Sí (Android adaptive icons)
